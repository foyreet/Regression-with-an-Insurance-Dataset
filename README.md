# Regression-with-an-Insurance-Dataset

https://www.kaggle.com/competitions/playground-series-s4e12/overview
Обучил модель `RandomForestRegressor` для точного предсказания стоимости страховки.
Реализовал пайплайн обработки данных: One-Hot кодирование, извлечение компонентов дат и генерацию сложных комбинированных признаков.
Результат по метрике **RMSLE** составил **1.18993**
